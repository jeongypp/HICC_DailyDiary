# HICC_DailyDiary
AI 피드백을 더한 나만의 일기장 웹 프로젝트 "하루한줄"입니다.

# 주요 기능(MVP)
1. 5개 영역(수면, 식사, 성취, 대인관계, 정서 안정)의 키워드에 기반하여 클릭 몇번으로 그날 하루의 일기를 숏폼 보는 시간 안에 완성.
2. AI 피드백을 받고 선택적으로 추가 메모 줄글 작성.
3. 일기장 관리: 그래프와 통계로 월별 감정 상태 추적 가능.

# 기술 스택
FE: React & Tailwind

BE: Spring boot & Java

DB: H2 Database / MySQL

API: Google Geminai 3.0 Flash API
